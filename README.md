# Next.js Product Management App

A full-stack product management application built with Next.js (App Router), NextAuth.js, MongoDB, and TailwindCSS.
The app includes public pages, protected dashboard routes, Google login, credentials login, and complete product CRUD operations.

# Features

# Authentication with NextAuth (Google + Credentials login)

# Protected Routes (Dashboard, Add Product, Manage Products)

# Product CRUD (Create, Read, Update, Delete)

# Modern UI with TailwindCSS

# Product Details Page with:

Large image/banner

Full description

Price, priority, date

Back button

# SweetAlert2 notifications

# REST API with MongoDB + Mongoose
